Удостовериться в возможности пользоваться sudo:
	sudo -v
	Если система запросит пароль и после его ввода не выдаст ошибку — значит, sudo доступен.
	какие команды вам разрешены: sudo -l

Выполнить базовые команды, которые позволят:
1) Узнать текущую нагрузку на ЦП и информацию о нем
uptime    # средняя нагрузка за 1, 5 и 15 минут
top       # интерактивный просмотр процессов и загрузки CPU (выход q)
htop      # улучшенная версия top (может отсутствовать, установите sudo apt install htop)
mpstat -P ALL 1 # загрузка каждого ядра в реальном времени (из пакета sysstat)

2) Узнать текущую нагрузку на RAM и информацию о ней
free -h                   # объём RAM и swap в человеко-читаемом виде
cat /proc/meminfo         # детальная информация о памяти

top                       # в строке "KiB Mem" видно использование
vmstat -s                 # статистика использования памяти
3) Узнать информацию о дисках
lsblk                     # дерево дисков и разделов (удобно)
sudo fdisk -l             # детальная информация о дисках (требует sudo)

df -h                     # свободное место на смонтированных разделах
du -sh /путь/к/папке      # размер конкретной папки

4) Узнать информацию о сетевых интерфейсах
ip a                      # современная команда (ip addr)
ifconfig                  # старая, но ещё используется (может требовать sudo apt install net-tools)
ip link show              # состояние интерфейсов
ss -tuln                  # открытые порты и соединения (аналог netstat)

5) Создать директорию, файл, записать в файл текст и вывести его в консоль
mkdir test_dir            # создать директорию test_dir
touch test_dir/myfile.txt # создать пустой файл внутри директории
echo "Привет, мир!" > test_dir/myfile.txt   # записать текст в файл (перезаписать)
echo "Ещё строка" >> test_dir/myfile.txt    # добавить строку в конец файла
cat test_dir/myfile.txt   # вывести содержимое файла в консоль

6) Узнать настройки имени и времени
hostname                  # текущее имя компьютера
hostnamectl               # подробная информация об имени и ОС (systemd)

date                      # текущая дата и время
timedatectl               # информация о часовом поясе, синхронизации времени (systemd)

Для изменения времени/даты
sudo date -s "2025-02-17 12:00"
